# FocusFlow AI - User Manual

This manual explains how to use the FocusFlow AI app from first launch to daily workflow.

## 1. What This App Does

FocusFlow AI helps students manage study life by combining:
- Tasks and deadlines
- Focus sessions (Pomodoro)
- AI-generated daily schedule
- Productivity analytics

## 2. First-Time Setup

### 2.1 Create an account
1. Open the app.
2. Select Register.
3. Enter your name, email, and password.
4. Complete the profile fields used for personalization.

### 2.2 Log in
1. Open the app.
2. Enter your email and password.
3. Tap Login.

## 3. Main Navigation

The app includes these primary areas:
- Home: Daily summary and quick actions
- Tasks: Create and manage assignments
- Focus: Start focus sessions
- Scheduler: Generate AI study plan
- Analytics: Review progress and trends
- Profile: Update account and preferences

## 4. Task Management

### 4.1 Add a task
1. Go to Tasks.
2. Tap Add Task.
3. Fill in title, deadline, difficulty, and estimated time.
4. Save the task.

### 4.2 Edit or complete a task
1. Open task details.
2. Change fields as needed.
3. Mark as completed when done.

### 4.3 Delete a task
1. Open task details.
2. Tap Delete.
3. Confirm.

## 5. Focus Sessions

### 5.1 Start a session
1. Go to Focus.
2. Choose task (optional).
3. Start timer.

### 5.2 During focus
- Pause/resume is available.
- End session when finished.

### 5.3 Session logging
Completed sessions are saved and used in analytics.

## 6. AI Scheduler

### 6.1 Generate schedule
1. Go to Scheduler.
2. Tap Regenerate Schedule.
3. Review generated blocks.

### 6.2 How it works
The scheduler uses:
- Task urgency and remaining workload
- User profile settings
- Predicted productivity windows
- Break recommendations

## 7. Analytics Dashboard

Use Analytics to monitor:
- Total focus time
- Completion rate
- Consistency and streaks
- Burnout risk and suggestions

## 8. Profile and Personalization

In Profile Settings you can update:
- Personal details
- Study preferences
- ML profile fields

These settings improve prediction quality and scheduling relevance.

## 9. Recommended Daily Workflow

1. Add or review tasks in the morning.
2. Regenerate schedule.
3. Run focus sessions throughout the day.
4. Mark completed tasks.
5. Check analytics at day end.

## 10. Troubleshooting

### 10.1 Cannot connect to server
- Confirm backend server is running.
- Confirm mobile and backend are on the same network for physical device testing.

### 10.2 Login fails
- Verify credentials.
- Re-register if account was not created successfully.

### 10.3 Schedule looks empty
- Make sure at least one active task exists.
- Regenerate schedule again after updating task data.
