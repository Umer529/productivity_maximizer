## 🚨 Important Notes

### Security
- ⚠️ Change JWT_SECRET in .env before production
- ⚠️ Don't commit .env with real secrets
- ⚠️ Use HTTPS in production
- ⚠️ Validate all user input
- ⚠️ Keep dependencies updated

### Performance
- 💡 ML predictions are cached for 5 minutes
- 💡 Database queries are fast (<5ms)
- 💡 Frontend uses lazy loading
- 💡 API uses pagination for large datasets

### Limitations
- Single-user per database (expand with multi-tenancy)
- No offline support (requires connection)
- Prayer times hardcoded (customize as needed)
- No real-time notifications (polling only)
